Bài 1
- HTML (Hyper Text Markup Language):
    là ngôn ngữ tiêu chuẩn để tạo nên trang web
    mô tả cấu trúc của một trang web


- HTML elements:
The <!DOCTYPE html> declaration defines that this document is an HTML5 document
The <html> : phần tử gốc của một trang HTML
The <head> : phần tử chứa thông tin meta của một trang HTML
The <title> : phần tử chỉ định tiêu đề (Được hiển thị trên tab trình duyệt)
The <body> : phần tử xác định phần thân và nó chứa tất cả các nội dung: headings, paragraphs, images, hyperlinks, tables, lists, etc.
The <h1> : phần tử xác định tiêu đề lớn
The <p> : phần tử xác định đoạn 

VD:
<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Phần tử HTML là tất cả mọi thứ từ tag mở đến tag đóng
Một số phần tử HTML không có nội dung (như phần tử <br>). Các phần tử này được gọi là phần tử rỗng. Các phần tử trống không có thẻ kết thúc!
